
Comandos básicos do git
	git init - dentro do diretório selecionado, inicia o monitoramento pelo git.
	git status - verifica a situação dos diretórios controlados. 
	git log - traz o log de todos os commits efetuados no diretório.

	rm -rf .git - remove o monitoramento do git, apagando o diretório de controle .git do diretório selecionado.


03 estágios do git

	Commited
		Gravado
		Significa que foi armazenado localmente com segurança
	Modified
		Modificado
		Arquivo foi alterado mas ainda não foi dado o "commit"
	Staged
		Selecionado
		Significa que marcou um arquivo modificado na versão corrente para ser parde do próximo commit

Workflow básico:
	Modifique os arquivos
	Selecione os arquivos que farão parte do próximo commit
	Faça o commit que utilizarão os arquivos "staged"


Andando entre commits/snapshots

	git checkout <commit id>
